HTML

Conceitos Básicos


Tags: São os blocos de construção do HTML. Exemplo: <h1>, <p>, <img>, <a>.

Elementos: Cada tag forma um elemento, que pode conter texto, atributos ou outros elementos.

Atributos: Adicionam informações extras às tags. Exemplo: src em <img src="imagem.jpg"> define o caminho da imagem.

Hierarquia: HTML é estruturado em forma de árvore, com elementos dentro de outros (como <div> contendo <p>).

Semântica: Tags como <header>, <nav>, <article> e <footer> ajudam a dar significado ao conteúdo, melhorando acessibilidade e SEO.

HTML, ou HyperText Markup Language, é a linguagem base da web. É com ela que criamos a estrutura de páginas, organizando o conteúdo em títulos, parágrafos, imagens, links, listas e muito mais.

Comece criando páginas simples com títulos, parágrafos e imagens.

Use um editor de código como VS Code para facilitar a escrita e visualização do HTML.

Pratique criando páginas pessoais, currículos online ou portfólios.
